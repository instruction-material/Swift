# SAD4 Map Places App

This project introduces maps and place-focused app design.

## Goals

- use `MapKit` and SwiftUI map views
- model places with coordinates and descriptive metadata
- connect list selection to a map presentation
- discuss permissions and map-centered UI design at a high level

## Starter Expectations

- wire list selection to map camera updates
- improve the place detail panel
- keep the structure ready for device testing later

## Solution Highlights

- map camera movement tied to selection
- markers for several places
- clearer detail presentation and polished copy
