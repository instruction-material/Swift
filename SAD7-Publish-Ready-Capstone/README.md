# SAD7 Publish-Ready Capstone

This project is the publish-ready capstone scaffold for the course.

## Goals

- combine navigation, lists, settings, and polish into one app shell
- think about scope, accessibility, screenshots, and metadata
- prepare students for TestFlight and App Store Connect workflow concepts

## Starter Expectations

- define scope and ship only a clear, coherent feature set
- complete the missing tabs and store-readiness checklist
- document accessibility and publishing considerations

## Solution Highlights

- a multi-tab capstone shell
- clearer state organization and settings
- a built-in release checklist and metadata-oriented mindset
