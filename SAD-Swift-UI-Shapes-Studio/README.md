# SADX SwiftUI Shapes Studio

This is an optional enrichment pack derived from the legacy `AD4` material.

It is intentionally separate from the main `SAD1`-`SAD7` sequence.

## Goals

- practice SwiftUI shapes such as `Circle`, `Ellipse`, and `Rectangle`
- build comfort with `Path` and arcs
- turn low-level shape primitives into a more expressive composed scene

## Pack Structure

- `starter/` gives a guided scaffold with visible TODOs
- `solution/` gives one polished reference implementation

## Enrichment Scope

This pack combines the three strongest ideas from the old `AD4` module:

1. classic shape composition
2. pie-chart and arc drawing
3. build-a-creature style composition

## Teaching Position

Use this pack only when a student would benefit from extra visual SwiftUI
practice or wants a creative detour. It should not replace the core course.
