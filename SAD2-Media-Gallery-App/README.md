# SAD2 Media Gallery App

This project turns basic SwiftUI layout into a richer media browsing experience.

## Goals

- work with arrays of model data
- build grid and detail layouts
- filter or favorite media items
- compare starter gallery logic with a fuller reference solution

## Starter Expectations

- finish category filtering
- improve the detail experience
- add lightweight interaction such as favorites

## Solution Highlights

- segmented filtering
- reusable cards
- detail presentation with clearer metadata
