# SAD6 API Reference App

This project introduces networking, decoding, loading states, and error states.

## Goals

- fetch JSON from a public endpoint
- decode API results into Swift types
- represent loading, loaded, and failed states in the UI
- build a simple reference-style app rather than a one-off fetch demo

## Starter Expectations

- improve the loading and error handling
- add search or filtering
- make the detail presentation clearer

## Solution Highlights

- async loading with refresh
- search filtering
- reusable row and detail views
