# SAD1 Welcome Profile App

This is the first code project for the expanded Swift app course.

## Goals

- understand the basic SwiftUI app entry point
- read and modify a simple view tree
- practice text, stacks, navigation, and reusable view sections
- build a beginner-friendly welcome/profile app that feels like a real app

## Starter Expectations

- improve the layout hierarchy
- add profile details and short goals
- make the secondary page feel intentional instead of placeholder-heavy

## Solution Highlights

- a polished welcome screen
- reusable section and tag views
- clean navigation to a detailed profile page
