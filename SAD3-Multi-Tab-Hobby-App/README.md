# SAD3 Multi-Tab Hobby App

This project introduces a multi-screen SwiftUI app shell with tabs and shared
state.

## Goals

- use `TabView` for high-level navigation
- keep small pieces of state in the right place
- share favorite selections across screens
- build a hobby app that feels broader than a single-page demo

## Starter Expectations

- connect the tab content so it shares useful state
- refine the schedule and favorites views
- make one tab act like a real dashboard

## Solution Highlights

- tab-based navigation
- favorite tracking across screens
- reusable cards and list rows
