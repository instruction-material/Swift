# SAD5 Simple Tracker App

This project covers lists, forms, CRUD workflows, and simple persistence.

## Goals

- create, update, and delete tracked items
- connect form inputs to list updates
- introduce `ObservableObject` and lightweight persistence
- show how an app becomes more useful once its data survives relaunches

## Starter Expectations

- finish editing and persistence
- keep item state changes predictable
- make the add flow and list behavior clear

## Solution Highlights

- a small store type
- `UserDefaults` persistence
- add/toggle/delete workflows with cleaner state ownership
